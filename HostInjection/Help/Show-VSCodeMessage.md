# Show-VSCodeMessage

## SYNOPSIS

Shows a message in Visual Studio Code.

## SYNTAX

### __AllParameterSets

```
Show-VSCodeMessage -Message <string> [-Items <string[]>] [-Modal] [-Type <string>] [-Wait]
 [-ResponseTimeout <int>] [<CommonParameters>]
```

## ALIASES

This cmdlet has no aliases.

## DESCRIPTION

Shows an information, warning, or error message in Visual Studio Code. Optional items are displayed as action buttons.

## EXAMPLES

### Example 1

```powershell
Show-VSCodeMessage -Message 'Build complete' -Type Information
```
Shows an informational message in Visual Studio Code.
