# Remove-VSCodeTextEditor

## SYNOPSIS

Removes a Visual Studio Code text editor reference.

## SYNTAX

### __AllParameterSets

```
Remove-VSCodeTextEditor -TextEditor <VsCodeTextEditor> [-Wait] [-ResponseTimeout <int>]
 [<CommonParameters>]
```

## ALIASES

This cmdlet has no aliases.

## DESCRIPTION

Removes or disposes a text editor reference managed by the PowerShell Pro Tools VS Code host.

## EXAMPLES

### Example 1

```powershell
$editor = Get-VSCodeTextEditor
Remove-VSCodeTextEditor -TextEditor $editor
```
Removes the active text editor reference.
