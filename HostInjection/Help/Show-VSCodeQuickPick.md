# Show-VSCodeQuickPick

## SYNOPSIS

Shows a quick pick list in Visual Studio Code.

## SYNTAX

### __AllParameterSets

```
Show-VSCodeQuickPick -Items <string[]> [-PlaceHolder <string>] [-CanPickMany] [-IgnoreFocusOut]
 [-Wait] [-ResponseTimeout <int>] [<CommonParameters>]
```

## ALIASES

This cmdlet has no aliases.

## DESCRIPTION

Shows a Visual Studio Code quick pick list and returns the selected item or items.

## EXAMPLES

### Example 1

```powershell
Show-VSCodeQuickPick -Items 'Debug','Release' -PlaceHolder 'Select configuration'
```
Prompts the user to select a build configuration.
