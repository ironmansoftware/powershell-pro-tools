# Get-VSCodeTerminal

## SYNOPSIS

Gets the active Visual Studio Code terminal.

## SYNTAX

### __AllParameterSets

```
Get-VSCodeTerminal [-Wait] [-ResponseTimeout <int>] [<CommonParameters>]
```

## ALIASES

This cmdlet has no aliases.

## DESCRIPTION

Returns the active Visual Studio Code terminal object so it can be used with commands such as Send-VSCodeTerminalText.

## EXAMPLES

### Example 1

```powershell
$terminal = Get-VSCodeTerminal
```
Gets the active terminal.
